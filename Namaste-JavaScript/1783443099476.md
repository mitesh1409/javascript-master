# #4 How functions work in JS ❤️ & Variable Environment | Namaste JavaScript Ep. 4

## 4.1 Code Example

Run the following code in the browser, check the different execution contexts and call stack.

```javascript
var x = 1;
foo();
bar();
console.log(x);

function foo() {
    var x = 10;
    console.log(x);
}

function bar() {
    var x = 100;
    console.log(x);
}
```
